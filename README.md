# StockPro App

StockPro is a stock market web application built using the MERN stack.

This project is inspired by platforms like Groww and was built as a practice project to improve my full stack development skills.

It allows users to explore stock market data, search stocks, and manage a virtual portfolio.

## Live Demo

https://stockpro-app-iota.vercel.app/

## Repository

https://github.com/DhirajKumar29-DK/stockpro-app

## Features

* User authentication
* Stock search
* Buy and sell stocks
* Portfolio management
* Wallet system
* Transaction history
* User profile management
* Dark and light mode

## Tech Stack

**Frontend**

* React.js
* Vite
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB Atlas

**API**

* Yahoo Finance API

## Deployment

* Frontend: Vercel
* Backend: Render
* Database: MongoDB Atlas

## Run Locally

```bash
git clone https://github.com/DhirajKumar29-DK/stockpro-app.git
npm install
npm run dev
```

This project was built for learning and practice to improve my MERN stack development skills and understand how stock trading applications work.
